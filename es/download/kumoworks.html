¿Qué es KumoWorks?

KumoWorks es un programa para crear paisajes con nubes. Permite generar imágenes de nubes a partir del dibujo de sus siluetas. Apuntamos a desarrollar una herramienta útil para crear fondos o pautas de color para producción de animación.
Repositorio GitHub de KumoWorks

[Uso básico]

  1. Ejecutar KumoWorks.
  2. Dibujar la silueta de una nube en el visor.
  3. Presionar la tecla Intro.
